5 Dunder files

What are files like `__main__.py`, or `__init__.py`:

__main__.py is a file is run,
when a command-line command like 

python dir_with_py_files
or 
python dir_with_py_files.zip
is run. general contents of __main__.py would be: 

#!/usr/bin/env python3

from .module import main

if __name__ == "__main__":
  main()
