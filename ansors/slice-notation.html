2 Slice notation

Some simple tools we have to slice are:

a[start:end] # items from 'start' through 'end-1'
a[start:]    # items from 'start' through the rest of the array
a[:end]    # items from the beginning through end-1
a[:]    # a copy of the whole array
a[start:end:step] # start through not past end, by step (unlike matlab, step isn't in the middle)
a[::-1]    # This would reverse the list, read from beginning to end with a step of -1

We can do slicing with negative indices


a[-1]    # get last item in the array
a[-1:]    # last item as an array
# a '-n' is just a shortcut to mean 'length(list)-n'
a[-4:-1:2]    # if len(a) == 4, then this line is equivalent to a[0:3:2]
a[-4:2]    # would be equivalent to a[0:2], you got the point right?
a[-1:2]    # this would be empty, remember python lists (or sets, tuples) are not cyclic, 
             # they just take two kinds of indices
