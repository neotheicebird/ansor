4 One Liners

This article is a compilation of cool logics that can be put together in one line

temperatures = [21, 20.5, 22.5, 26, 31, 38, 36, 32, 40]
for T in temperatures:
    if T > 37:
        print "Temperature exceeded"
        break
else:
    print "Within limits"

Can we do this in one line? whoah... that would look bad Or is it? check this line

print "Temperature exceeded" if next((T for T in temperatures if T > 37), False) else "Within limits"

"next(generator with an if condition)" is an alternate for any "for > if > break" logic
