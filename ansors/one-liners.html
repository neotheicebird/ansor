4 One Liners

This article is a compilation of cool logics that can be put together in one line

first element in an iterable that meets a condition: 
temperatures = [21, 20.5, 22.5, 26, 31, 38, 36, 32, 40]
for T in temperatures:
    if T > 37:
        print "Temperature exceeded"
        break
else:
    print "Within limits"

Can we do this in one line? whoah... that would look bad won't it Or is it? :D checkout this one-liner

print "Temperature exceeded" if next((T for T in temperatures if T > 37), False) 
                            else "Within limits"

"next(generator with an if condition)" is an alternate for any "for > if > break" logic

We have shuffled x in range(0, 10) all shuffled and y = sin(x) would give corresponding sine values. The problem comes when we plot x, y. The plot looks all scrambled and distorted, when we expect a neat sine wave. To solve this is easy. All we have to do is sort(x) and reorder y in the same way x is sorted. For example, if the first element in list x is moved 3 steps to right, then the first element of y is moved 3 steps to right. Here is an one liner: 

[x for (y,x) in sorted(zip(Y,X))]
